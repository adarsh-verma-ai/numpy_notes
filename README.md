# NumPy Tutorial 📘

A beginner-friendly NumPy tutorial notebook covering the fundamentals of numerical computing in Python.

This project demonstrates how to create, manipulate, and perform operations on NumPy arrays using practical examples inside a Jupyter Notebook.

---

# 🚀 Topics Covered

The notebook includes hands-on examples for:

* Introduction to NumPy
* Creating NumPy arrays
* Array properties and attributes
* Array operations
* Indexing and slicing
* Data types in NumPy
* Multi-dimensional arrays
* Mathematical operations
* Aggregation functions
* Broadcasting
* Array reshaping
* Working with 2D and 3D arrays

# 📌 Sample Concepts Demonstrated

## Creating Arrays

```python
import numpy as np

arr = np.array([1, 2, 3, 4, 5])
print(arr)
```

## Array Operations

```python
print(arr + 5)
print(arr * 2)
```

## Multi-Dimensional Arrays

```python
arr2D = np.array([[1,2,3],[4,5,6]])
print(arr2D.shape)
```
